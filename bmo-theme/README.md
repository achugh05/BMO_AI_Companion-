HOW TO INSTALL------------------------------------------------------------

Ensure ES-DE is installed on your system
Navigate to your ES-DE installation folders (on Windows go to Windows(C:)/Users/USER/ES-DE)
Drag and drop the whole folder called bmo-theme into your Themes folder
Congrats you have successfully installed the BMO theme!
